Sobre el desarrollo

El objetivo de este proyecto es desarrollar un sistema completamente funcional. sitio web de comercio electrónico que utiliza el marco de desarrollo: Angular, TypeScript, HTML, CSS, .Net Core, Bootstrap, iconos FontAwesome, Angular Material y API JSON. El El sitio web tiene como objetivo realizar un demo de lo que seria la implementación de un E-Commerce por un equipo de trabajo experiencia.

Lenguajes Usados
  • TypeScript
  • Angular Framework
  • HTML
  • CSS
  • Bootstrap
  • JSON API
  • .NET Core
